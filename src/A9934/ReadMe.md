완전 이진 트리
---

date : 2022-02-07\
url : https://www.acmicpc.net/problem/A9934

예제
--

### 1)
- input
```
2
2 1 3
```

- output
```
1
2 3
```

### 2)

- input
```
3
1 6 4 3 5 2 7
```

- output
```
3
6 2
1 4 5 7
```

풀이
---

```
8
4 12
2 6 10 14
1 3 5 7 9 11 13 15 
```
마지막 줄은 2씩, 그 다음 줄은 4씩 값이 2의 제곱근 만큼 값이 증가가 되는 규칙이 있습니다.
